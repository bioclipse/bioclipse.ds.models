Carcinogenicity Endpoint

A carcinogen is a type of mutagen that specifically leads to cancer.

Data source

Carcinogenicity data originating from the Carcinogenic Potency Database (CPDB) [1] was downloaded from the distributed structure-searchable toxicity (DSSTox) public database network [2] versioned 5d (20 Nov 2008). The dataset contained 892 chemical structures, and TD50 values for rat was chosen as response property.
