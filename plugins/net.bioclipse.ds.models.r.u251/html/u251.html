u251 cell line model

A model was built on data from a cell line screening, where cells derived from a malignant glioblastoma tumour (U251).

Data source

Data was downloaded from the NCI60 cell line anticancer drug screen [1]. The dataset contained 3515 chemical structures.

Models

Model type Model implementation Comment
QSAR SVM model with Signature descriptor height 1-3 implemented in R
